Programming for GIA Core Skills: Assessment 1 breakdown

This assessment focussed on the principles of agent based modelling. The final submission outlines the steps taken to create agent based model accompanied with documentation advising on best ways to execute the code. Links presented in the assessment 1 page will direct you to Github where files are available for download

Programming for GIA Core Skills: Assessment 2 breakdown

This assessment focused on creating an advanced version of the agent-based modelling theory encountered in assessment 1. The aim for this assignment was to get a series of 25 drunks to navigate around a 300x300 environment grid, stopping when they had reached their home. Following this the brief required a section of code which would write a file depicting the density of the drunks travel patterns. Links presented in the assessment 2 page will direct you to Github where files are available for download

Web-based GIS: Assessment 1 breakdown

This assessment focused on learning the basics of web design. Each topic introduces vital coding languages/markup languages including PHP,HTML and Javascript. Topics also utilised a combination of google maps and leaflet to help display the utility of GIS and coding working holistically.

Web-based GIS: Assessment 2 breakdown

This assessment featured a leaflet map displaying data regarding key landmarks involved in the 1st through to the 4th Crusades. The data was handcrafted by myself by researching the landmarks, adding information regarding the landmarks geographical location and finally uploading the data to PGSQL. Following this through the use of PHP and Javascript the data was presented on a leaflet map.
